# Estructura base

Esta carpeta contiene el encarpetado principal del dominio siguiendo `Screaming Architecture`.

## Modulos base

- `panel`
- `instituciones`
- `periodos`
- `carga-academica`
- `horarios`
- `nomina`
- `pendientes`
- `examenes`
- `documentos`
- `drive`

## Nota temporal

La carpeta `app/` del proyecto se mantiene en la raiz por ahora para no mezclar la definicion arquitectonica con una migracion tecnica temprana.

Cuando el proyecto avance a la siguiente fase, se puede evaluar mover `app/` a `src/app` junto con el resto de la estructura.
